🧩 Mosaico: Composición de Grupos Estudiantiles con K-means (2023)¶

¿Qué es? Investigación aplicada que resuelve un problema concreto de la gestión escolar: ¿cómo formar grupos de estudiantes que sean pedagógicamente óptimos? El sistema combina criterios académicos, variables sociales y aleatorización controlada para lograr composiciones equilibradas y diversas. Tesina de Magíster en Ciencias de Datos (UdeC).

Impacto: Optimización pedagógica de la composición de grupos, reduciendo el sesgo en la formación manual y mejorando la equidad en el aula.

¿Cómo está construido? Aplicación de clustering K-means con Python (scikit-learn) para modelar composición óptima considerando múltiples variables simultáneamente.

Tecnologías: Python · scikit-learn · K-means · Análisis estadístico · Jupyter
